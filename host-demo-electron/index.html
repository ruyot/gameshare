🎮 Game Stream Host

How to Stream Your Game:

  1. Start the signaling server (in a separate terminal): cd signaling && node index.js
  2. Build the streamer (if not already built): cd host_streamer && cargo build --release
  3. Click "Start Streaming" below to launch the streamer
  4. Copy the OFFER that appears in the logs
  5. On your Mac, open the client.html file and paste the OFFER to connect
